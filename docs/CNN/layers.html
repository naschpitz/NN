Layer Documentation

Detailed reference for all layer types in the CNN library. Each page covers the mathematical operation, propagation, backpropagation, and implementation details.

🧩 Conv2D

Convolutional layer with configurable filters, strides, and padding.

⚡ ReLU

Rectified Linear Unit activation function.

📏 Pool

Max and average pooling for spatial downsampling.

📊 Normalization

Instance and Batch normalization for training stability.

📐 Flatten

Reshape 3D tensor to 1D vector for dense layers.

🌐 GlobalAvgPool

Global average pooling — reduces each channel to its spatial mean.

🌐 GlobalDualPool

Concatenates global avg and global max per channel (doubles channel count).

🔄 Residual

Skip connections with identity or 1×1 projection shortcut.
