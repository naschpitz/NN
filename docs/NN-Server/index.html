NN-Server — Neural Network Inference Server

A lightweight HTTP server for serving ANN and CNN model predictions over a REST API. NN-Server loads a trained model into a thread-safe pool and exposes a POST /predict endpoint that accepts JSON vectors or raw image uploads, returning predictions as JSON or PNG images.

🤖 Auto-Detect

Automatically identifies ANN vs CNN from the model JSON

🔀 Concurrent

Thread-safe core pool handles multiple requests in parallel

🖼️ Image Input

Send JPEG/PNG images directly — auto-resized to model input shape

⚡ Lightweight

No external web framework — pure Qt TCP server

📊 JSON & Image Output

Returns predictions as JSON arrays or PNG images

How It Works

  1. Load a trained model — provide a .nnmodel package (tar archive with model.json + params.bin) from NN-CLI training output
  2. NN-Server detects whether it's an ANN or CNN model and creates a pool of Core instances
  3. Accept HTTP requests — clients POST to /predict with JSON or image data
  4. Run inference — a Core is acquired from the pool, prediction runs, result is returned
  5. Return output — JSON array for classification, or PNG image for image-to-image models

Quick Example

# Create a config.json
{
  "model": "trained_model.json",
  "port": 8080
}

# Start the server
./NN-Server config.json

# Send an image for classification
curl -X POST http://localhost:8080/predict \
     -H "Content-Type: image/jpeg" \
     --data-binary @photo.jpg
