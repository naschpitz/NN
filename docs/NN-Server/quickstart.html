Quick Start & Build

Prerequisites

RequirementMinimumNotes
C++ CompilerC++17GCC 9+, Clang 10+
CMake3.14
Qt6Core + Network + Concurrent modules
OpenCL1.2Optional — required only for GPU-trained models
A trained model—JSON output from NN-CLI training

Clone & Build

# Clone the monorepo with all submodules
git clone --recursive https://github.com/naschpitz/NN.git
cd NN

# Build NN-Server (requires CMakeUserPresets.json at repo root)
cd NN-Server
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . -j$(nproc)

# Or build all components from the repo root:
# ./build.sh

# The NN-Server executable is ready
ls -la NN-Server
Monorepo structure: NN-Server links CNN and ANN as sibling directories, with OpenCLWrapper under ANN/extern/OpenCLWrapper and CNN/extern/OpenCLWrapper. The --recursive flag clones all submodules.

Verify Installation

# Create a config.json file
cat > config.json <<EOF
{
  "model": "/path/to/trained_model.json"
}
EOF

# Start the server
./NN-Server config.json

# Expected output:
# NN-Server starting...
#   Config:    config.json
#   Model:     /path/to/trained_model.json
#   Port:      8080
#   Pool size: 8
#   Output:    vector (JSON)
#
# Listening on port 8080
# Ready to accept requests.

# In another terminal, check the health endpoint:
curl http://localhost:8080/health
# {"status":"ok"}

Run Tests

Integration tests are included. They start the server automatically with a bundled test model and send concurrent HTTP requests.

cd build
./test_endpoints

# Expected output:
# === NN-Server Endpoint Integration Tests ===
#
# Starting NN-Server...
# Server is ready.
#   testHealth...
#   testNotFound...
#   testPredictBadJson...
#   testPredictMissingInput...
#   testPredictImageSingle...
#   testPredictImageConcurrent (5 different images)...
#   testPredictImageRepeatedConcurrent (10x same image)...
#
# === Results: 44 passed, 0 failed ===

A separate unit-test binary covers the logger:

./test_logger
# All logger tests passed.

Project Structure

NN-Server/
├── main.cpp                      # Entry point, config.json parsing
├── NN-Server_HttpServer.hpp/cpp  # HTTP server & request handler
├── NN-Server_CorePool.hpp/cpp    # Thread-safe pool of ANN/CNN Cores
├── NN-Server_Loader.hpp/cpp      # Model loading & network detection
├── NN-Server_ImageLoader.hpp/cpp # Image decode/encode via stb
├── NN-Server_Logger.hpp/cpp      # Request logging with circular write
├── NN-Server_NetworkType.hpp     # NetworkType enum (ANN/CNN)
├── libs/
│   ├── nlohmann/                 # JSON header-only library
│   └── stb/                      # stb image headers (read/write/resize)
├── extern/
│   └── CNN/                      # CNN submodule (includes ANN)
├── tests/
│   ├── test_main.cpp             # Test entry point
│   ├── test_endpoints.cpp        # Integration tests
│   ├── test_logger.cpp           # Logger unit tests
│   ├── test_helpers.hpp          # Test utilities (CHECK, HTTP helpers)
│   └── fixtures/                 # Test model & sample images
└── docs/                         # This documentation
