ANN — Artificial Neural Network Library

A production-ready, template-based C++ library for building, training, and deploying fully-connected (dense) neural networks on CPU and GPU.

Overview

ANN is a self-contained C++ library that implements a classic multi-layer perceptron (MLP) with support for ReLU, Sigmoid, and Tanh activation functions. It can run on CPU with multi-threaded training via Qt Concurrent, or on one or more GPUs using OpenCL for massively parallel computation.

Key Features

  • Factory pattern — Core<T>::makeCore() creates the right backend (CPU or GPU) from configuration
  • Template-based — Supports int, float, double with explicit instantiations
  • Multi-threaded CPU — Thread-local accumulators eliminate lock contention during training
  • Multi-GPU — Automatic gradient merging across devices with per-GPU progress reporting
  • He / Xavier initialisation — Correct weight initialisation for ReLU and Sigmoid/Tanh
  • Pure library — Accessible entirely via C++ function calls; no built-in file I/O
  • Step-by-step API — predict → backpropagate → accumulate → update for custom training loops
  • Progress callbacks — Real-time epoch/sample/loss reporting
