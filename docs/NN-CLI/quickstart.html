Quick Start & Build

Prerequisites

RequirementMinimumNotes
C++ CompilerC++17GCC 9+, Clang 10+, MSVC 2017+
CMake3.14
Qt6Core + Concurrent modules
OpenCL1.2Optional — required only for --device gpu

Clone & Build

# Clone the monorepo with all submodules
git clone --recursive https://github.com/naschpitz/NN.git
cd NN/NN-CLI

# Build
mkdir build && cd build
cmake ..
make -j$(nproc)

# Run tests
./test_nncli

# The NN-CLI executable is ready
./NN-CLI --help
Monorepo structure: NN-CLI is part of the NN monorepo. CNN and ANN are sibling directories, not submodules. OpenCLWrapper is a git submodule under both ANN/extern/OpenCLWrapper and CNN/extern/OpenCLWrapper. The --recursive flag clones the submodules.

Verify Installation

# Check that it runs
./build/NN-CLI --help

# NN-CLI - Neural Network Command Line Interface (ANN + CNN)
#
# Usage:
#   NN-CLI --config <file> --mode train [options]                       # Training
#   NN-CLI --config <file> --mode predict --input <f>                   # Predict (batch)
#   NN-CLI --config <file> --mode test [options]                        # Evaluation
#   NN-CLI --config <file> --mode calibrate --id-images <dir> [opts]   # OOD threshold
#
# Options:
#   --config, -c <file>    Path to JSON configuration file (required)
#   --mode, -m <mode>      Mode: 'train', 'predict', 'test', or 'calibrate'
#   --device, -d <device>  Device: 'cpu' or 'gpu' (overrides config file)
#   --input, -i <file>     Path to JSON file with batch inputs (predict mode, required)
#   --input-type <type>    Input data type: 'vector' or 'image' (overrides config file)
#   --samples, -s <file>   Path to JSON file with samples (train/test modes)
#   --idx-data <file>      Path to IDX3 data file (alternative to --samples)
#   --idx-labels <file>    Path to IDX1 labels file (requires --idx-data)
#   --output, -o <file>    Output file/dir (default: predict_<input>.json or threshold.json)
#   --output-type <type>   Output data type: 'vector' or 'image' (overrides config file)
#   --log-level, -l <lvl>  Log level: quiet, error, warning, info, debug (default: error)
#   --gpu-profile          Enable OpenCL GPU kernel profiling (adds ~12% overhead)
#
# Calibrate-mode options:
#   --id-images <dir>      Directory of in-distribution images (recursed) [required]
#   --ood-dir <dir>        OOD images directory (default: <cwd>/extern-datasets/ood)
#   --id-sample-count <N>  Random subsample size for ID set (default 500)
#   --ood-sample-count <N> Random subsample size for OOD set (default 1500)
#   --id-percentile <P>    ID percentile used as the threshold (default 95)
#   --no-fetch             Don't auto-download OOD if --ood-dir is empty (default: fetch)
#   --help, -h             Show this help message

First Training Run

Here's the fastest way to verify everything works — train a small ANN:

# 1. Create a minimal config file
cat > tiny_config.json << 'EOF'
{
  "layers": [
    { "numNeurons": 2, "actvFunc": "relu" },
    { "numNeurons": 4, "actvFunc": "relu" },
    { "numNeurons": 1, "actvFunc": "sigmoid" }
  ],
  "train": {
    "numEpochs": 100,
    "batchSize": 64,
    "learningRate": 0.1
  }
}
EOF

# 2. Create sample data (XOR problem)
cat > xor_samples.json << 'EOF'
{
  "samples": [
    { "input": [0,0], "output": [0] },
    { "input": [0,1], "output": [1] },
    { "input": [1,0], "output": [1] },
    { "input": [1,1], "output": [0] }
  ]
}
EOF

# 3. Train!
./build/NN-CLI -c tiny_config.json -m train -s xor_samples.json -l info

├── NN-CLI_ANNLoader.hpp/cpp     # ANN-specific config & sample loading
├── NN-CLI_CNNLoader.hpp/cpp     # CNN-specific config & sample loading
├── NN-CLI_DataSplitter.hpp/cpp  # Stratified validation splitter
├── NN-CLI_Runner.hpp/cpp        # Core runner base class
├── NN-CLI_ANNRunner.hpp/cpp     # ANN training/predict/test runner
├── NN-CLI_CNNRunner.hpp/cpp     # CNN training/predict/test runner
├── NN-CLI_CalibrateRunner.hpp/cpp # Calibrate runner

├── NN-CLI_AugmentationConfig.hpp # Augmentation config struct

├── NN-CLI_ValidationDatasetConfig.hpp # Validation dataset config
├── NN-CLI_TrainingSummary.*     # Training summary table

├── NN-CLI_LossReferenceTable.*  # Loss / confidence reference table

├── NN-CLI_TrainingProfiler.*    # Training profiler

├── NN-CLI_RunnerUtils.hpp       # Runner utility helpers
├── NN-CLI_RunnerObserver.hpp    # Runner observer interface
├── NN-CLI_CalibrateUtils.*      # Calibration utilities
├── libs/

├── examples/
│   ├── MNIST/                   # ANN & CNN MNIST example configs
│   └── ISIC-MILK10k/            # ISIC skin lesion example
└── tests/                       # Integration tests
