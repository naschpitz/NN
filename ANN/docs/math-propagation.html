Forward Propagation

Forward propagation computes the network's output by passing the input through each layer sequentially.

On this page

  1. Algorithm
  2. Dropout
  3. Implementation

Algorithm

Given an input vector, the forward pass proceeds layer by layer from input to output:

  1. Input layer (\(l = 0\)): Set activations directly from the input: \[a^{(0)} = \text{input}\]
  2. Hidden and output layers (\(l = 1, \ldots, L\)): For each layer, compute the pre-activation and apply the activation function: \[z^{(l)} = W^{(l)} \cdot a^{(l-1)} + b^{(l)}\] \[a^{(l)} = f^{(l)}\!\left(z^{(l)}\right)\]

The final layer's activations \(a^{(L)}\) are the network's output (prediction).

Dropout

During training, dropout randomly zeroes out a fraction of neuron activations to prevent overfitting. For each hidden layer (not the output layer):

\[ \text{mask}_j \sim \text{Bernoulli}(1 - p) \qquad\qquad a_j^{(l)} \leftarrow a_j^{(l)} \cdot \frac{\text{mask}_j}{1 - p} \]

Where \(p\) is the dropout rate. The scaling by \(\frac{1}{1-p}\) (inverted dropout) ensures that the expected value of each activation remains unchanged, so the network can be used at inference time without modification.

During inference (predict/test mode), dropout is not applied — all neurons are active.

Backpropagation with dropout: The same mask is applied to the gradients during the backward pass. If a neuron was dropped during forward propagation, its gradient is also zeroed out. See Backpropagation.

Implementation

ANN_CoreCPUWorker.cpp — propagate()

// Set input layer activations
for (ulong j = 0; j < inputSize; j++)
  this->actvs[0][j] = input[j];

// Forward pass through hidden and output layers
for (ulong l = 1; l < numLayers; l++) {
  // Compute z = W * a + b
  for (ulong j = 0; j < numNeurons; j++) {
    this->zs[l][j] = this->parameters.biases[l][j];
    for (ulong k = 0; k < prevNumNeurons; k++)
      this->zs[l][j] += this->parameters.weights[l][j][k] * this->actvs[l - 1][k];
  }

  // Apply activation function
  ActvFunc::calculate(zsData, actvsData, numNeurons, actvFuncType, false);

  // Apply dropout (training only, hidden layers only)
  if (applyDropout && l < numLayers - 1 && dropoutRate > 0) {
    T scale = 1 / (1 - dropoutRate);
    for (ulong j = 0; j < numNeurons; j++) {
      T mask = bernoulli(1 - dropoutRate) ? scale : 0;
      this->dropoutMasks[l][j] = mask;
      this->actvs[l][j] *= mask;
    }
  }
}
