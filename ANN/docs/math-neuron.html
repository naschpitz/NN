The Neuron Model

A single artificial neuron computes a weighted sum of its inputs, adds a bias, and passes the result through a non-linear activation function.

On this page

  1. Weighted Sum
  2. Activation
  3. Implementation

Weighted Sum

For neuron \(j\) in layer \(l\), the pre-activation (also called the "z-value") is:

\[ z_j^{(l)} = b_j^{(l)} + \sum_{k} w_{jk}^{(l)} \cdot a_k^{(l-1)} \]

Where:

  • \(w_{jk}^{(l)}\) — weight connecting neuron \(k\) in layer \(l-1\) to neuron \(j\) in layer \(l\)
  • \(a_k^{(l-1)}\) — activation of neuron \(k\) in the previous layer
  • \(b_j^{(l)}\) — bias for neuron \(j\)

Activation

The pre-activation is then passed through a non-linear activation function to produce the neuron's output:

\[ a_j^{(l)} = f\!\left(z_j^{(l)}\right) \]

The choice of \(f\) depends on the layer configuration. See Activations for the available functions and their derivatives.

Implementation

ANN_CoreCPUWorker.cpp — propagate()

The forward pass iterates over all neurons in each layer, computing the weighted sum and then applying the activation function for the entire layer at once:

// Compute z = b + W * a for all neurons in layer l
for (ulong j = 0; j < numNeurons; j++) {
  this->zs[l][j] = this->parameters.biases[l][j];

  for (ulong k = 0; k < prevNumNeurons; k++)
    this->zs[l][j] += this->parameters.weights[l][j][k] * this->actvs[l - 1][k];
}

// Apply activation function to the entire layer
ActvFunc::calculate(zsData, actvsData, numNeurons, actvFuncType, false);
Note: Softmax is a special case — it operates on the entire layer's z-values at once (to compute the normalization denominator), unlike ReLU/Sigmoid/Tanh which are element-wise. The ActvFunc::calculate() function handles this distinction internally.
