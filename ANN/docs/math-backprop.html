Backpropagation

Backpropagation computes the gradient of the loss with respect to every weight and bias in the network, using the chain rule applied layer by layer from output to input.

Overview

The backward pass proceeds from the output layer to the first hidden layer. At each layer \(l\), three things are computed:

  1. \(\partial L / \partial z^{(l)}\) — how the loss changes with respect to the pre-activations (using the activation derivative)
  2. \(\partial L / \partial W^{(l)}\) and \(\partial L / \partial b^{(l)}\) — the weight and bias gradients (used by the optimizer to update parameters)
  3. \(\partial L / \partial a^{(l-1)}\) — the gradient propagated to the previous layer

Output Layer Gradient

The starting point is the gradient of the loss function with respect to the output activations:

\[ \frac{\partial L}{\partial a_j^{(L)}} = \text{(depends on loss function — see Loss Functions)} \]

Activation Derivative

The chain rule converts the activation gradient into a pre-activation gradient:

\[ \frac{\partial L}{\partial z_j^{(l)}} = f'^{(l)}\!\left(z_j^{(l)}\right) \cdot \frac{\partial L}{\partial a_j^{(l)}} \]

For softmax, this is handled differently — see Softmax backward.

Weight and Bias Gradients

Once \(\partial L / \partial z_j^{(l)}\) is known, the weight and bias gradients follow directly:

Weight gradient

\[ \frac{\partial L}{\partial w_{jk}^{(l)}} = a_k^{(l-1)} \cdot \frac{\partial L}{\partial z_j^{(l)}} \]

The gradient of a weight is the activation of the neuron it connects from, multiplied by the pre-activation gradient of the neuron it connects to.

Bias gradient

\[ \frac{\partial L}{\partial b_j^{(l)}} = \frac{\partial L}{\partial z_j^{(l)}} \]

The bias gradient is simply the pre-activation gradient itself (since \(\partial z / \partial b = 1\)).

Propagate to Previous Layer

To continue the backward pass, the gradient must be propagated to the previous layer's activations:

\[ \frac{\partial L}{\partial a_k^{(l-1)}} = \sum_{j} w_{jk}^{(l)} \cdot \frac{\partial L}{\partial z_j^{(l)}} \]

Each neuron \(k\) in layer \(l-1\) receives gradient contributions from every neuron \(j\) in layer \(l\) that it connects to, weighted by the corresponding connection weight.

Dropout in Backpropagation

If dropout was applied during the forward pass, the same mask is applied to the gradients:

\[ \frac{\partial L}{\partial a_j^{(l)}} \leftarrow \frac{\partial L}{\partial a_j^{(l)}} \cdot \text{mask}_j \]

Neurons that were dropped during forward propagation receive zero gradient — they do not participate in the parameter update for that sample.

Implementation

ANN_CoreCPUWorker.cpp — backpropagate()

// Output layer: compute dL/da from cost function
for (ulong j = 0; j < numNeurons; j++)
  this->dCost_dActvs[l][j] = calc_dCost_dActv(j, output);

// For each layer from output to first hidden:
for (ulong l = L; l >= 1; l--) {
  // Apply dropout mask to gradients
  if (dropoutMasks[l] not empty)
    dCost_dActvs[l][j] *= dropoutMasks[l][j];

  // dL/dz = f'(z) * dL/da
  ActvFunc::calculate(zs, actvs, numNeurons, actvFuncType, true,
                      dCost_dActvs, dCost_dZs);

  // Compute weight and bias gradients
  dCost_dWeights[l][j][k] = actvs[l-1][k] * dCost_dZs[j];
  dCost_dBiases[l][j]     = dCost_dZs[j];

  // Propagate gradient to previous layer
  dCost_dActvs[l-1][k] = Σ_j weights[l][j][k] * dCost_dZs[j];
}
