Mathematical Foundations

This section covers the mathematical foundations of the CNN library. Each page connects the mathematical formulas to the exact C++ functions and GPU kernels that implement them.

  • Pooling — Max and average pooling backpropagation
  • Activations — ReLU and its gradient
  • Instance Normalization — Per-sample spatial normalisation, forward/backward pass, and comparison with Batch Normalization
  • Weight Initialization — He/Kaiming initialisation for ReLU networks
  • Optimizers — SGD and Adam update rules
  • Gradient Flow — End-to-end gradient flow through a complete CNN pipeline
