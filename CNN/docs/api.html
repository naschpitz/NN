PredictResult<T> CNN_PredictResult.hpp

Returned by predict(). Carries both the post-activation output and the pre-activation (z) values of the ANN dense head's last layer. Logits enable calibration / out-of-distribution detection scores (max-logit, logit-norm, free-energy −log Σ exp(zᵢ)) that softmax discards because softmax is shift-invariant: softmax(z) = softmax(z + c).

FieldTypeDescription
outputOutput<T>Post-activation values of the dense head's last layer (e.g. softmax probabilities)
logitsLogits<T>Pre-activation values (z) of the dense head's last layer
NCHW Layout: data[c * H * W + h * W + w]. Channels are contiguous — all height×width values for channel 0 come first, then channel 1, etc.

4. Core<T> — Abstract Base Class

Defined in CNN_Core.hpp. All CNN operations go through this interface.

MethodReturnsDescription
makeCore(config)unique_ptr<Core<T>>Factory: creates CoreCPU or CoreGPU
predict(ulong numSamples, const InputProvider<T>&)PredictResults<T>Virtual: streaming batch propagation. Pulls inputs in chunks from provider, runs forward propagation on each batch, and bounds peak host memory to one batch regardless of total dataset size. Mirrors the SampleProvider pattern used by train() / test(). Each result carries output (post-activation) and logits (pre-activation z of the dense head's last layer). Distributes work across threads / GPUs per batch.
predict(input)PredictResult<T>Single-input convenience. Delegates to the streaming overload with a one-shot provider. The underlying ANN sub-core uses the step worker directly (no threading) to avoid deadlocks when called from within CNN's thread pool.
train(numSamples, sampleProvider)voidFull training loop with SGD
test(numSamples, sampleProvider)TestResult<T>Evaluate loss on a test set using on-demand sample loading
setTrainingCallback(cb)voidAttach progress callback
getParameters()const Parameters<T>&Get current conv + dense parameters
getTrainingMetadata()TrainingMetadata<T>Timing and loss from last train()
getPredictMetadata()PredictMetadata<T>Timing from last predict()
