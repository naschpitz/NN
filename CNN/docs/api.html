1. Types & Aliases

TypeDefinitionDescription
Shape3Dstruct { ulong c, h, w; }3D tensor shape (channels, height, width)
Tensor3D<T>struct { Shape3D shape; vector<T> data; }3D tensor in flat NCHW layout
Tensor1D<T>std::vector<T>1D tensor
Input<T>Tensor3D<T>Network input (3D image tensor)
Output<T>std::vector<T>Network output (1D classification vector)
Sample<T>struct { Input<T> input; Output<T> output; }Training/test sample
Samples<T>std::vector<Sample<T>>Collection of samples

Shape3D Methods

MethodReturnsDescription
size()ulongReturns c × h × w
operator==()boolEquality comparison

Tensor3D<T> Methods

MethodDescription
at(c, h, w)Element access at channel c, row h, column w
size()Returns total number of elements
resize(newShape)Resize data to new shape
resize(newShape, value)Resize and fill with value
fill(value)Fill all elements with value
NCHW Layout: data[c * H * W + h * W + w]. Channels are contiguous — all height×width values for channel 0 come first, then channel 1, etc.

3. Configuration Structs

CoreConfig<T> CNN_CoreConfig.hpp

Top-level configuration struct. Contains all settings needed to create and run a CNN.

FieldTypeDescription
modeTypeModeTypeTRAIN, PREDICT, or TEST
deviceTypeDeviceTypeCPU or GPU
numThreadsintCPU threads (0 = all cores)
numGPUsintGPUs to use (0 = all available)
inputShapeShape3DInput tensor shape (channels, height, width)
layersConfigLayersConfigCNN + dense layer architecture
costFunctionConfigCostFunctionConfig<T>Cost function type and optional weights
trainingConfigTrainingConfig<T>Training hyperparameters (epochs, learning rate, batch size)
testConfigTestConfigTest evaluation settings (batch size)
parametersParameters<T>Pre-trained parameters (optional)
progressReportsulongProgress callback frequency for all modes (default 1000)
logLevelLogLevelLog verbosity level (default ERROR)

TrainingConfig<T> CNN_TrainingConfig.hpp

FieldTypeDescription
numEpochsulongTotal training epochs (default 0)
learningRatefloatLearning rate (default 0.01). Used as the base learning rate α for both SGD and Adam.
batchSizeulongMini-batch size (default 64)
shuffleSamplesboolShuffle sample order each epoch (default true)
dropoutRatefloatDropout probability for dense hidden layers during training (default 0.0 = disabled). Uses inverted dropout: activations are scaled by 1/(1−p) during training so no adjustment is needed at inference. Applied only to the dense (fully-connected) layers, not convolutional layers.
optimizerOptimizer<T>Optimizer configuration (default: SGD)

TestConfig CNN_TestConfig.hpp

Configuration for test evaluation. Separate from TrainingConfig to allow independent batch sizing.

FieldTypeDescription
batchSizeulongMini-batch size for test evaluation (default 64). Controls how many samples are loaded into memory at once during testing.

Optimizer<T> CNN_Optimizer.hpp

FieldTypeDescription
typeOptimizerTypeOptimizer algorithm: SGD (default) or ADAM
beta1TExponential decay rate for first moment estimate (default 0.9). Only used by Adam.
beta2TExponential decay rate for second moment estimate (default 0.999). Only used by Adam.
epsilonTSmall constant for numerical stability (default 10⁻⁸). Only used by Adam.

4. Core<T> — Abstract Base Class

Defined in CNN_Core.hpp. All CNN operations go through this interface.

MethodReturnsDescription
makeCore(config)unique_ptr<Core<T>>Factory: creates CoreCPU or CoreGPU
predict(input)Output<T>Propagation — CNN → Flatten → ANN
train(numSamples, sampleProvider)voidFull training loop with SGD
test(numSamples, sampleProvider)TestResult<T>Evaluate loss on a test set using on-demand sample loading
setTrainingCallback(cb)voidAttach progress callback
getParameters()const Parameters<T>&Get current conv + dense parameters
getTrainingMetadata()TrainingMetadata<T>Timing and loss from last train()
getPredictMetadata()PredictMetadata<T>Timing from last predict()

5. CoreCPU<T>

CPU implementation using the ANN library as the dense-layer backend.

Internal MethodDescription
propagateCNN(input, training, intermediates, poolMaxIndices)Run input through all CNN layers. When training=true, stores intermediate activations for backpropagation
backpropagateCNN(dCNNOut, intermediates, poolMaxIndices, dFilters, dBiases)Backpropagate through CNN layers
initializeConvParams()He initialisation for conv filters
resetCNNAccumulators()Zero gradient accumulators
accumulateCNNGradients()Add sample gradients to accumulators
updateCNNParameters(numSamples)SGD update for conv parameters

6. CoreGPU<T> & CoreGPUWorker<T>

CoreGPU orchestrates multiple CoreGPUWorker instances for multi-GPU training.

CoreGPU<T>

MethodDescription
initializeWorkers()Create one CoreGPUWorker per GPU
mergeCNNGradients()Sum CNN accumulators across all GPUs
mergeANNParameters()Average dense-layer parameters across GPUs

CoreGPUWorker<T>

MethodDescription
predict(input)Single-sample inference on GPU
trainSubset(samples, start, end, ...)Train on a range of samples
testSubset(samples, start, end)Evaluate a range of samples
readAccumulatedGradients()Read gradient buffers from GPU
setAccumulators()Write merged gradients back to GPU
update(numSamples)Run SGD update kernels

7. Layer Classes

All layer classes are stateless — they expose only static methods.

Conv2D<T>

MethodReturnsDescription
propagate(input, config, params)Tensor3D<T>Convolution with padding
backpropagate(dOut, input, config, params, dFilters, dBiases)Tensor3D<T>Compute gradients for filters, biases, and input

ReLU<T>

MethodReturnsDescription
propagate(input)Tensor3D<T>Element-wise max(0, x)
backpropagate(dOut, input)Tensor3D<T>Pass gradient where input > 0

Pool<T>

MethodReturnsDescription
propagate(input, config, maxIndices)Tensor3D<T>Max or average pooling; stores max indices
backpropagate(dOut, inputShape, config, maxIndices)Tensor3D<T>Route/distribute gradient back through pool

InstanceNorm<T>

MethodReturnsDescription
propagate(input, inputShape, params, config, training, batchMean*, batchVar*, xNormalized*)Tensor3D<T>Normalize per channel per sample over spatial dimensions; computes per-sample spatial stats for both training and inference
backpropagate(dOut, inputShape, params, config, batchMean, batchVar, xNormalized, dGamma, dBeta)Tensor3D<T>Compute gradients for γ, β, and input

Flatten<T>

MethodReturnsDescription
propagate(input)Tensor1D<T>Reshape 3D → 1D (zero-copy)
backpropagate(dOut, inputShape)Tensor3D<T>Reshape 1D → 3D
Template instantiations: All classes are explicitly instantiated for int, float, and double.
