CNN + ANN Pipeline

The complete inference pipeline chains CNN layers (convolutional, activation, pooling) with ANN dense layers. Data flows as a 3D tensor through CNN, flattens to 1D, then flows through ANN.

The pipeline consists of two phases:

  1. CNN Phase: Input 3D tensor flows through Conv2D, ReLU, Pool layers in sequence. Each layer outputs a 3D tensor that becomes the input to the next layer.
  2. ANN Phase: The final 3D tensor is flattened to 1D, then passed to ANN's dense layers for final classification/regression.
