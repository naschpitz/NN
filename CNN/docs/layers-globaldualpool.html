GlobalDualPool Layer

Global Dual Pooling concatenates the results of Global Average Pooling and Global Max Pooling for each channel, producing a richer spatial summary that captures both the mean activation and the strongest activation per channel.

On this page

  1. Purpose
  2. Propagation
  3. Backpropagation
  4. Parameters

Purpose

  • Richer representation — captures both average and peak activations, giving the dense layers two complementary views of each feature
  • Better classification — max pooling highlights the strongest feature response (is this pattern present?) while avg pooling captures overall feature density (how much of this pattern is present?)
  • Still input-size agnostic — like GlobalAvgPool, works with any spatial dimensions

Propagation

For input with \(C\) channels and spatial size \(H \times W\), the output has \(2C\) values (shape \(2C, 1, 1\)):

\[ \text{output}[c] = \frac{1}{HW} \sum_{h,w} \text{input}[c, h, w] \quad \text{(avg, first C values)} \] \[ \text{output}[C + c] = \max_{h,w} \text{input}[c, h, w] \quad \text{(max, second C values)} \]

The output doubles the channel count: if the input has 128 channels, the output has 256 elements. This must be accounted for in the subsequent flatten and dense layer sizing.

Backpropagation

Each pooling component backpropagates independently, and their gradients are summed at the input:

\[ \frac{\partial L}{\partial x_{c,h,w}} = \underbrace{\frac{1}{HW} \frac{\partial L}{\partial y_c^{avg}}}_{\text{avg gradient}} + \underbrace{\frac{\partial L}{\partial y_c^{max}} \cdot \mathbf{1}[x_{c,h,w} = \max]}_{\text{max gradient}} \]

The max gradient flows only to the spatial position that held the maximum value for that channel. The avg gradient is distributed evenly across all spatial positions.

Parameters

No learnable parameters. GlobalDualPool is a fixed aggregation operation.

JSON Configuration

{ "type": "globaldualpool" }
Output size: If the last convolutional block produces \(C\) channels, GlobalDualPool outputs \(2C\) elements. The first dense layer must accept \(2C\) inputs.
