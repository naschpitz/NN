GlobalAvgPool Layer

Global Average Pooling reduces each channel's spatial dimensions (H × W) to a single value by computing the spatial mean. The output shape is (C, 1, 1), collapsing all spatial information into a per-channel summary.

On this page

  1. Purpose
  2. Propagation
  3. Backpropagation
  4. Parameters

Purpose

  • Replaces flatten + dense — eliminates the massive parameter count from flattening large spatial feature maps into a dense layer
  • Spatial invariance — the mean is independent of the exact spatial arrangement of activations
  • Regularization effect — fewer parameters means less overfitting compared to fully-connected alternatives
  • Input-size agnostic — works with any spatial dimensions, enabling inference on different image sizes than training

Propagation

For each channel \(c\), compute the mean over all spatial positions:

\[ \text{output}[c] = \frac{1}{H \times W} \sum_{h=0}^{H-1} \sum_{w=0}^{W-1} \text{input}[c, h, w] \]

The operation modifies the tensor in-place, compacting the data to the first \(C\) elements and setting the shape to \((C, 1, 1)\).

Typical placement: GlobalAvgPool is placed after the last convolutional block and before flatten. It replaces the traditional stack of max pools that progressively shrink spatial dimensions.

Backpropagation

The gradient is distributed evenly across all spatial positions in each channel:

\[ \frac{\partial L}{\partial x_{c,h,w}} = \frac{1}{H \times W} \frac{\partial L}{\partial y_c} \]

The gradient tensor is expanded from \((C, 1, 1)\) back to the original \((C, H, W)\) shape.

Parameters

No learnable parameters. GlobalAvgPool is a fixed aggregation operation.

JSON Configuration

{ "type": "globalavgpool" }
