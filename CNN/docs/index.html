CNN — Convolutional Neural Network Library

A high-performance, template-based C++17 library for building, training, and running convolutional neural networks. CNN supports both CPU and GPU (OpenCL) execution with multi-GPU training.

🧩 Layer Types

Conv2D, ReLU, Pool (Max & Avg), Flatten — compose any architecture

⚡ GPU Accelerated

OpenCL kernels for every layer — propagation & backpropagation

🔗 ANN Integration

Dense layers delegated to ANN library for full pipeline

📐 Template-based

Works with int, float, double

Overview

The CNN library implements a complete convolutional neural network pipeline: 3D tensor input passes through configurable convolutional, activation, and pooling layers, then flattens into a 1D vector for dense layers (handled by the ANN library). The library handles propagation, backpropagation through all layer types, gradient accumulation, and parameter updates (SGD or Adam optimizer).

Key Features

  • NCHW tensor layout — channels-first for optimal convolution performance
  • Padding strategies — VALID (no padding), SAME (preserve spatial size), FULL
  • Configurable strides — independent Y and X strides for convolution and pooling
  • He initialisation — proper weight initialisation for ReLU networks
  • Multi-GPU training — gradient merging + parameter averaging across GPUs
  • Factory pattern — transparent CPU/GPU backend selection
  • Pure library — accessible entirely via C++ function calls; no built-in file I/O
