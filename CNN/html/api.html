1. Types & Aliases

TypeDefinitionDescription
Shape3Dstruct { ulong c, h, w; }3D tensor shape (channels, height, width)
Tensor3D<T>struct { Shape3D shape; vector<T> data; }3D tensor in flat NCHW layout
Tensor1D<T>std::vector<T>1D tensor
Input<T>Tensor3D<T>Network input (3D image tensor)
Output<T>std::vector<T>Network output (1D classification vector)
Sample<T>struct { Input<T> input; Output<T> output; }Training/test sample
Samples<T>std::vector<Sample<T>>Collection of samples

Shape3D Methods

MethodReturnsDescription
size()ulongReturns c × h × w
operator==()boolEquality comparison

Tensor3D<T> Methods

MethodDescription
at(c, h, w)Element access at channel c, row h, column w
size()Returns total number of elements
resize(newShape)Resize data to new shape
resize(newShape, value)Resize and fill with value
fill(value)Fill all elements with value
NCHW Layout: data[c * H * W + h * W + w]. Channels are contiguous — all height×width values for channel 0 come first, then channel 1, etc.

ValueDescription

ModeType

ValueDescription

LayerType

ValueDescription

POOLPooling layer (max or average)

SlidingStrategyType

ValuePaddingEffect
VALID0No padding — output shrinks
SAME⌊kernel/2⌋Output same size as input (stride=1)
FULLkernel − 1Maximum padding — output grows

PoolTypeEnum

ValueDescription

AVGAverage pooling (distributes gradient evenly)

4. Core<T> — Abstract Base Class

Defined in CNN_Core.hpp. All CNN operations go through this interface.

MethodReturnsDescription
makeCore(config)unique_ptr<Core<T>>Factory: creates CoreCPU or CoreGPU
predict(input)Output<T>Forward pass — CNN → Flatten → ANN
train(samples)voidFull training loop with SGD
test(samples)TestResult<T>Evaluate loss on a test set
setTrainingCallback(cb)voidAttach progress callback
getParameters()const Parameters<T>&Get current conv + dense parameters
getTrainingMetadata()TrainingMetadata<T>Timing and loss from last train()
getPredictMetadata()PredictMetadata<T>Timing from last predict()

5. CoreCPU<T>

CPU implementation using the ANN library as the dense-layer backend.

Internal MethodDescription
forwardCNN(input)Run input through all CNN layers
forwardCNN(input, intermediates, poolMaxIndices)Forward pass storing all intermediate activations (for training)
backwardCNN(dCNNOut, intermediates, poolMaxIndices, dFilters, dBiases)Backpropagate through CNN layers
initializeConvParams()He initialisation for conv filters
resetCNNAccumulators()Zero gradient accumulators
accumulateCNNGradients()Add sample gradients to accumulators
updateCNNParameters(numSamples)SGD update for conv parameters

6. CoreGPU<T> & CoreGPUWorker<T>

CoreGPU orchestrates multiple CoreGPUWorker instances for multi-GPU training.

CoreGPU<T>

MethodDescription
initializeWorkers()Create one CoreGPUWorker per GPU
mergeCNNGradients()Sum CNN accumulators across all GPUs
mergeANNParameters()Average dense-layer parameters across GPUs

CoreGPUWorker<T>

MethodDescription
predict(input)Single-sample inference on GPU
trainSubset(samples, start, end, ...)Train on a range of samples
testSubset(samples, start, end)Evaluate a range of samples
readAccumulatedGradients()Read gradient buffers from GPU
setAccumulators()Write merged gradients back to GPU
update(numSamples)Run SGD update kernels

7. Layer Classes

All layer classes are stateless — they expose only static methods.

Conv2D<T>

MethodReturnsDescription
predict(input, config, params)Tensor3D<T>Forward convolution with padding
backpropagate(dOut, input, config, params, dFilters, dBiases)Tensor3D<T>Compute gradients for filters, biases, and input

ReLU<T>

MethodReturnsDescription
predict(input)Tensor3D<T>Element-wise max(0, x)
backpropagate(dOut, input)Tensor3D<T>Pass gradient where input > 0

Pool<T>

MethodReturnsDescription
predict(input, config, maxIndices)Tensor3D<T>Max or average pooling; stores max indices
backpropagate(dOut, inputShape, config, maxIndices)Tensor3D<T>Route/distribute gradient back through pool

Flatten<T>

MethodReturnsDescription
predict(input)Tensor1D<T>Reshape 3D → 1D (zero-copy)
backpropagate(dOut, inputShape)Tensor3D<T>Reshape 1D → 3D
Template instantiations: All classes are explicitly instantiated for int, float, and double.
