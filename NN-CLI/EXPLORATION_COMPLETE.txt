KEY FINDINGS
================================================================================

BATCH NORMALIZATION
  Location: extern/CNN/CNN_BatchNorm.hpp/cpp
  Status: ✅ Fully implemented
  Features:
    - Training mode: computes batch statistics, updates running averages
    - Inference mode: uses running statistics
    - Stores intermediates (mean, var, normalized) for backprop
    - Per-channel scale (gamma) and shift (beta)
  API: propagate() and backpropagate() static methods

CONVOLUTION (Conv2D)
  Location: extern/CNN/CNN_Conv2D.hpp/cpp
  Status: ✅ Fully implemented
  Features:
    - Standard 2D convolution with stride and padding
    - Supports VALID and SAME padding strategies
    - Multiple filters with per-filter bias
    - Naive implementation (no im2col optimization)
  API: propagate() and backpropagate() static methods

COST FUNCTIONS
  Location: *_CostFunctionConfig.hpp and *_Worker.cpp
  Status: ✅ Three types available
  Types:
    1. SQUARED_DIFFERENCE (MSE)
    2. WEIGHTED_SQUARED_DIFFERENCE
    3. CROSS_ENTROPY
  Features:
    - Available in both ANN and CNN
    - String conversion helpers
    - Per-output weights support
    - Numerical stability (epsilon = 1e-7)

================================================================================
INTEGRATION POINTS
================================================================================

All components integrate through the Worker class:
  - Forward pass: propagate() methods called in sequence
  - Loss calculation: calculateLoss() uses configured cost function
  - Backward pass: backpropagate() methods called in reverse order
  - Parameter updates: Gradients accumulated for optimizer

Parameter management via unified Parameters<T> struct:
  - convParams: One per convolution layer
  - bnParams: One per batch norm layer
  - denseParams: Dense layer parameters (delegated to ANN)
